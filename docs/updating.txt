1. Clear browser cache

2. stop server
sudo a2dissite <apache site name>
sudo /etc/init.d/apache2 reload

3. Clear server cache
python manage.py clear_cache

4. clear tilestache cache
sudo rm -rf /tmp/<slugified app name>-stache

5. Optional, precache some urls and tiles
python manage.py precache

6. Reload site
sudo a2ensite <apache site name>
sudo /etc/init.d/apache2 reload
